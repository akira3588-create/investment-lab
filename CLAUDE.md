# Claude Security Guidelines

## 承認ルール

### ファイル操作・コマンド実行
- ファイルの作成・編集・削除を行う前に、内容と理由を明示し、ユーザーの承認（y/n）を必ず求める
- コマンド実行前に、実行内容と影響範囲を説明し、承認を求める

### 外部アクセス
- 外部サイト・APIへのアクセスが発生する場合、アクセス先URLと目的を明示し、承認を求める

## 機密情報の取り扱い

- IDやパスワード、APIキーなどの機密情報はコードに直接書かない
- 機密情報は `.env` ファイルに記載し、環境変数として使用する
- `.env` ファイルは `.gitignore` に登録し、GitHubにアップロードしない
- `.env.example` にキー名のみ記載してテンプレートとして管理する
